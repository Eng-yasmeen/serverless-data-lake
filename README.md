# Serverless Data Lake on AWS

## Overview
A serverless data lake built using Terraform on AWS.

## Services Used
- S3
- Glue
- Athena
- CloudWatch
- SNS
- IAM
- GitHub Actions

## Data Flow
S3 → Glue → Athena → Query & Analytics

## CI/CD
GitHub Actions runs Terraform validation and plan on every push.

## Status
Project is fully working and deployed.
